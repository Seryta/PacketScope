PacketScope 是一款 Android 端只读 PCAP（抓包文件）查看器，设计用来配合
PCAPdroid 抓包，提供 Wireshark 风格的分析界面：包列表 + 协议树 + 字节级
Hex 视图，三者联动高亮。

## 功能

- <b>协议解析</b>：Ethernet / Linux SLL / Raw IP / IPv4 / IPv6 / ICMP /
  ICMPv6 / TCP / UDP / DNS / HTTP / TLS（含 SNI）/ QUIC
- <b>TCP 会话分析</b>：相对 seq/ack、Retransmission / Out-of-Order /
  Duplicate ACK 标记
- <b>TCP 重组 + HTTP body 解析</b>：跨 segment HTTP 请求 / 响应，支持
  Content-Length / chunked / gzip / deflate
- <b>TLS 解密</b>：提供 SSLKEYLOGFILE 后可解 TLS 1.2（AES-GCM +
  ChaCha20-Poly1305）和 TLS 1.3（AES-128-GCM / AES-256-GCM /
  ChaCha20-Poly1305）
- <b>QUIC v1 Initial 解密</b>：payload + CRYPTO frame 提取（无需 keylog）
- <b>Wireshark 风格过滤</b>：tcp / port 53 / host 1.1.1.1 / sni example.com
  / http.method POST / dns.name google ... + and / or / not / 括号
- <b>PCAPdroid 集成</b>：分享 / ACTION_VIEW 打开 .pcap 文件，解 PCAPdroid
  app 归属 trailer，实时接入 UDP Exporter 流
- <b>大文件支持</b>：mmap 路径加载，单文件上限 500 MB

## 隐私

PacketScope 离线运行，不收集 / 记录 / 传输 / 共享任何数据。唯一的网络
访问是绑定本地 UDP socket 接收 PCAPdroid 实时流。详见仓库 PRIVACY.md。

## 与 PCAPdroid 搭配

PCAPdroid（同样在 F-Droid 上）在 Android 上无 root 抓包，导出为 .pcap
文件（用分享打开 PacketScope）或实时 UDP 流（PacketScope 监听可配端口，
默认 1234）。
